Use image from Raspberry Archive. Extract zip file and do "dd if=2013-09-10-wheezy-raspbian.img of=/dev/sdb bs=4M" to transfer it to the 2GB SD card you just inserted into your Linux PC. Make very sure it really is /dev/sdb! Now pop it into your Raspberry Pi Model B. Check your router for its DHCP IP address. SSH to it using pi:raspberry as credentials ("ssh IP -l pi").

I used wget to get the files from this page. Image files go to /home/pi/atv/images, sound.wav goes into /home/pi/atv/sounds, Python script into /home/pi/atv. Now do "sudo su -" to become root. config.txt goes into /boot/, rc.local into /etc/. Do "chmod +x rc.local" to make it executable. Reboot to get the testpattern. See also Milatz' (PA3BWE) site. Especially for the switches to select patterns (ground pin #24 and #26). I did not have to install python-pygames, samba or RPi-GPIO. Worked right away. Fancy that! I did heavily hack the original Python script (atv3.py) though.

NOTE: Do NOT use the June 2014 Wheezy image. It sucks (no SSH?!?). Wasted hours to figure it out...
